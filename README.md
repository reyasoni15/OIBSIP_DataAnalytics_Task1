# OIBSIP Data Analytics Internship – Task 1

## Task: Exploratory Data Analysis (EDA) on Retail Sales Data

### Objective
Perform Exploratory Data Analysis on retail sales data to uncover trends, patterns, and business insights that can support data-driven decision-making.

### Steps Performed
- Loaded and cleaned the retail sales dataset
- Handled missing values and inconsistent data
- Analyzed sales trends over time
- Identified top-performing products and categories
- Visualized revenue distribution and seasonal patterns
- Derived actionable business insights from the data

### Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook

### Outcome
Identified key sales trends, top-performing product categories, and seasonal patterns in the retail data, along with practical insights that could help improve business and marketing strategy.
